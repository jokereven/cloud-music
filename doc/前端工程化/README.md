## 前端工程化

```
前端工程化是使用软件工程的技术和方法来进行前端的开发流程、技术、工具、经验等规范化、标准化，
其主要目的为了提高开发过程中的开发效率，减少不必要的重复工作时间。
```

### 研发流程
```
技术选型

开发
本地测试
代码提交
编译、打包、构建
部署
集成测试
发布上线
监控运维
```
